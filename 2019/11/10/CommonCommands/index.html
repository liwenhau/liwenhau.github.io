Hexo常用命令
2019-11-10

hexo clean

清空缓存文件

hexo generate 或 hexo g

生成网站静态文件到public 文件夹

hexo deploy 或 hexo d

自动生成网站静态文件，并部署到仓库

hexo server 或 hexo s

启动本地服务器，预览主题

hexo new page about

新建一个标题为 about 的页面，默认链接地址为 主页地址/about/

hexo new "新文章"

新建一个标题为 新文章 的页面，文章标题日期可以随时修改
