一.安装hexo-tag-aplayer

在你的blog的根目录下安装 hexo-tag-aplayer
npm install --save hexo-tag-aplayer

二.使用

直接放在你想要放的文章中

标签参数

  • title : 曲目标题
  • authhor: 曲目作者
  • url: 音乐文件 URL 地址
  • picture_url: (可选) 音乐对应的图片地址
  • narrow: (可选) 播放器袖珍风格
  • autoplay: (可选) 自动播放，移动端浏览器暂时不支持此功能
  • width:xxx: (可选) 播放器宽度 (默认: 100%)
  • lrc:xxx: (可选) 歌词文件 URL 地址

三.使用网易云资源

我歌曲和歌词文件使用的都是网易云的

1.获取歌曲资源

打开网易云音乐，找到你打算下载的歌，然后截取歌曲链接的id
https://music.163.com/#/song?id=451703096

拼接获取歌曲的链接
http://music.163.com/song/media/outer/url?id=歌曲的ID.mp3

然后复制到浏览器打开可以直播播放，Ctrl+S 可以下载
这个就是歌曲的地址咯

2.获取歌词资源

获取歌词也是，找到你要找的歌，然后截取歌曲链接的id
https://music.163.com/#/song?id=451703096

拼接歌词链接
http://music.163.com/api/song/media?id=歌曲的ID

大功告成 哈哈哈！ 是不是很简单
